Chapter 1: Understanding the Basics

In this chapter, you'll learn what HTML is, how it forms the structure of a webpage, and how to create your first HTML page.


<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>First HTML Page</title>
    </head>
    <body>
        <h1>Hello, world!</h1>
        <p>Welcome to HTML basics.</p>
    </body>
</html>
